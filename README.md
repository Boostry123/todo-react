first steps:
1) npm init y
2) make an env file as following:

  PORT=5000
  
  DB_PORT=5432
  
  DB_PASSWORD={your db password}
 
  TZ = {your time zone}

To start the React App:
npm start

To start the server:
npm run dev

App created with:
HTML , CSS , React , TypeScript , Postgresql

Also used:
moment(for date managing)
Dnd-kit (Making the components dragable)
